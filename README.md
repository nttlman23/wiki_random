# Случайная Википедия

Небольшой сайт для просмотра случайных статей из Википедии. По кнопке «Следующая статья» загружается новая случайная страница.

## Запуск

Откройте `index.html` в браузере или запустите локальный сервер:

```bash
npx --yes serve .
```

Затем откройте адрес, который выведет команда (обычно http://localhost:3000).

## Как это работает

Сайт использует [MediaWiki Action API](https://www.mediawiki.org/wiki/API:Random) (`generator=random`) для выбранного языка Википедии (по умолчанию — русский).
